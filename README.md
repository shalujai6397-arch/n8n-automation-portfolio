# AI Automation & Analytics Portfolio
I design and deploy high-performance automation ecosystems using **n8n orchestration** and **Power BI dashboards**. 
My expertise lies in transforming manual business processes into automated, data-driven workflows that save time and eliminate human error. From building autonomous **AI Agents** to real-time inventory tracking, I provide end-to-end technical solutions for modern businesses.

## 🚀 Featured AI & Automation Projects

### 1. Autonomous Lead Qualification Agent
**Automated Sales & CRM Enrichment**
* **The Solution:** An end-to-end sales automation that captures inbound leads, applies **AI-driven scoring logic**, and enriches prospect data automatically.
* **Core Logic:** Uses structured output parsing and conditional branching to prioritize high-value leads for CRM entry.
* **Impact:** Dramatically reduces manual sales research time and ensures 24/7 lead response.

### 2. Multimodal AI Telegram Agent
**Voice, Vision, and Text Orchestration**
* **The Solution:** A sophisticated agent that processes **Voice (OpenAI Whisper)**, **Images (GPT-4o Vision)**, and text to provide human-like interactions within Telegram.
* **Core Logic:** Implemented contextual memory and complex routing in **n8n** to handle different media types seamlessly.
* **Impact:** Demonstrates the ability to build production-ready AI assistants that "see" and "hear."

---

### 3. Telegram Inventory Stock Agent
**Automated Inventory Management via n8n**
* **The Challenge:** A business needed a way to check and update stock levels instantly without opening a laptop or manual sheets.
* **The Solution:** Built an AI-driven agent that connects **Telegram** to **Airtable** via **n8n**. Users can simply type a product name in Telegram to get real-time stock updates.
* **Tech Stack:** n8n, Telegram Bot API, Airtable, JavaScript.
